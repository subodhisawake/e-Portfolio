Project Goal & Overview

This project focused on the foundational skill of deconstructing and modifying existing code. Starting with a generic HTML and JavaScript quiz template, the goal was to completely customize both the functionality (JavaScript) and the presentation (HTML) to create an interactive, themed FIFA World Cup Quiz.

View Live Quiz Deployment (Vercel)

Technical Deconstruction: JavaScript (popsolution.js)

The core logic of the quiz was modified to handle the new thematic content and question count. Key changes to the `popsolution.js` file included:

  • Questions Updated: I replaced all original questions with five new questions related to the FIFA World Cup, covering topics such as tournament history, top scorers, and host nations.
  • Answer Choices and Solutions: Each question’s multiple-choice options and corresponding correct answers were meticulously updated to match the new FIFA-themed content.
  • Compliments Modified: The compliment system was retained, ensuring it continues to provide positive, randomized feedback when the user answers correctly.
  • Total Questions Variable: The variable that controls the quiz length was updated to match the new content:
    // Original total question count variable
    var total = 6;

Structural and Thematic Modification: HTML (index.html)

The original HTML structure was preserved to ensure JavaScript compatibility but was heavily customized for clarity and thematic presentation. Below are the specific updates made to the HTML file:

  • Page Title and Metadata: Updated the title tag for a more descriptive and thematic page name.
  • Page Structure and Layout: I retained the necessary div and table structure for the JavaScript interaction but added a custom caption heading in the caption tag to replace the generic "JavaScript Kit Pop Quiz."
  • Interactive Elements: The core interactive elements—the answer dropdown, along with the Next and Back buttons—were tested and confirmed to correctly trigger the dynamic JavaScript functions. The “Cheat Mode” checkbox and “Start Over” link were kept functional.
  • Embedded JavaScript and File Linking: The external JavaScript file was properly linked, and the inline JavaScript section (essential for quiz generation) was retained and validated.
  • Readability and Formatting: The HTML code was consistently formatted and indented for improved readability and easier debugging.

Deployment and Testing

After local development and testing, the final quiz was prepared for external access:

  • Web Hosting: The project folder (containing `index.html` and `popsolution.js`) was uploaded to a public repository on GitHub named `fifa-quiz`.
  • Deployment Platform: The quiz was deployed on Vercel, ensuring it is accessible via a live web link and functions correctly across various browsers.
  • Compatibility: The HTML file was validated to ensure no syntax errors or broken tags existed prior to deployment.
