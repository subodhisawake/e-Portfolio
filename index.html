This e-Portfolio serves as a showcase for the projects and assignments completed during this course. I'm passionate about web development, and I aim to use these projects to demonstrate my growing understanding of the course material and my ability to apply theoretical concepts to practical challenges. Please explore my work below!

Course Projects

Project 1: Bitmap and Vector Imagery

This project showcases a foundational understanding of image formats by developing two distinct assets: a digital book cover using bitmap imagery (raster) and a professional logo using vector imagery (scalable).

Project 2: HTML and JavaScript Quiz (FIFA World Cup Edition)

This assignment involved deconstructing and modifying an existing quiz script. I customized the core JavaScript file (setting `total` to 6, updating all questions/solutions) and the HTML structure to create an interactive, FIFA World Cup-themed quiz. The project demonstrates front-end code modification and deployment via Vercel.

Project 3: Drag-n-Drop Assessment (Football Player Positions)

This project required customizing a script for a drag-and-drop assessment. I modified the app's topic, converting it into a Football Player Positions quiz. Key modifications involved modernizing the CSS using Flexbox for responsiveness and troubleshooting deployment challenges on Vercel.

Project 4: E-Portfolio Template (The Course Homepage)

This project is the foundational e-Portfolio itself, designed using HTML and a linked external CSS file to provide a single, consistent, and responsive homepage for navigating all course assignments.

Project 5: Educational Video ("The Caste System: Separating Myth from Reality")

This is an artistic digital production utilizing still images, a voiceover, and background music to create an informational video on The Caste System. It demonstrates consistent use of the Ken Burns effect (zoom/pan) and cross-fade transitions.
